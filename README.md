# *robosys2026
ロボットシステム学授業用課題1。やり直しました


![test](https://github.com/misujumpei/robosys2026/actions/workflows/test.yml/badge.svg)

コマンドライン引数で「関数名（sin/cos/tan）」と「角度（度数法）」を受け取り、三角関数の計算結果を出力するコマンドです。
誤った入力（数値以外や引数不足など）に対しては、適切なエラーメッセージを表示します。

## インストール方法
以下のコマンドを入力してください。

```bash
$git clone [https://github.com/misujumpei/robosys2026.git$](https://github.com/misujumpei/robosys2026.git$) cd robosys2026
```

以下のコマンドを入力し実行権限を付与してください。
```bash
$ chmod +x trig
```


## 引数のルール
本コマンドを使用する際は、以下のルールに従って引数を指定してください。

### 1. 第1引数: 関数名
使用できる関数は以下の**3種類**です。これ以外を指定するとエラーになります。
* `sin` : 正弦（サイン）
* `cos` : 余弦（コサイン）
* `tan` : 正接（タンジェント）

### 2. 第2引数: 角度
角度は**度数法（degree）の数値**で入力してください。
* 整数（例: `90`）または小数（例: `45.5`）が使用可能です。
* 文字列（例: `abc`）や記号を入力するとエラーになります。


## 実行例
 ### 基本的な使い方
  コマンドライン引数として、**コマンド本体**、**関数名**、**角度** の3つをスペース区切りで入力し実行します。

  **書式:**

  $ ./trig [関数] [角度]

### ※計算精度に関する注意
 コンピュータの浮動小数点演算の仕様上、計算結果に微小な誤差が含まれることがあります。これはバグではありません。

 **誤差の例:**
 `sin 720` は数学的には `0` ですが、計算上は「0に限りなく近い値（e-16など）」が出力される場合があります。

```bash
$ ./trig sin 720
-4.898587196589413e-16
```

## 使用例
 **sin（正弦）の計算例:**
 90度を指定すると、正しく 1.0 が出力されます。

```
$ ./trig sin 90
1.0
```

###エラー時の動作
 入力が不適切な場合は、Pythonのエラーをそのまま出さず、ユーザーに分かりやすいメッセージを表示して終了します。

####数値以外が入力された場合:
 角度に文字や記号が含まれていると、再入力を促すメッセージを出します。
 **実行例:**
 ```bash
 $ ./trig sin abc
 エラー: 角度には数値を入力してください
 ```

####未対応の関数を指定した場合:
 対応していない関数名が入力された場合も、エラーを検知します。
 **実行例:**
 $ ./trig log 10
 エラー: log は未対応です
